# 2026年3月25日の作業記録

## 実装内容
Shopifyからエクスポートされた注文データ（`CSVデータ.xlsx`）を読み込み、商品別に集計して指定のテンプレート形式（`G2828-02　きなこがでろーんTシャツ.xlsx`）に変換するシステムを構築しました。

### 主な機能
1.  **データの自動抽出と集計**:
    *   商品名（Lineitem name）から「商品名」「色」「サイズ」を自動的にパース。
    *   SKUベースで商品をグループ化し、色・サイズ別の数量を合計。
2.  **テンプレートへの自動書き込み**:
    *   既存のExcelテンプレートのレイアウトを維持したまま、集計データを所定のセルに書き込み。
    *   `S`, `M`, `L`, `XL`, `XXL`, `XXXL` のサイズマッピングに対応。
    *   商品コード（SKU）の表示形式を自動調整（例: `G2828-002` -> `G2828-02`）。
3.  **複数商品の同時処理**:
    *   `CSVデータ.xlsx` に含まれる全商品を対象に、個別の発注書ファイルを生成。
4.  **StreamlitによるUI構築**:
    *   ブラウザからExcelファイルをアップロードし、変換結果をZIP形式で一括ダウンロードできるインターフェースを実装（`app.py`）。

### 使用技術
*   Python
*   Pandas (データ集計)
*   Openpyxl (Excelファイルの編集・保存)
*   Streamlit (Webインターフェース)

## 成果物
