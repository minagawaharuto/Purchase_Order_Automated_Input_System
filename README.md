# 発注書自動作成システム (Purchase Order Automated Input System)

Shopifyからエクスポートされた注文データ（Excel形式）を解析し、特定の商品ごとに集計された発注書（Excelファイル）を自動生成するシステムです。

## 概要
このシステムは、Shopifyの注文エクスポートデータ（`CSVデータ.xlsx`）を読み込み、商品名、色、サイズ、数量を抽出・集計します。集計結果は、あらかじめ用意されたエクセルのテンプレート（`G2828-02　きなこがでろーんTシャツ.xlsx`）の指定されたセルに自動的に流し込まれ、商品ごとの発注書として保存されます。

## 主な機能
- **自動集計**: 商品名（`Lineitem name`）から商品名、色、サイズを分離し、数量を合計します。
- **テンプレート維持**: 既存のエクセルテンプレートのレイアウト（罫線、フォント、固定文言など）を保ったまま数値を更新します。
- **複数商品対応**: CSVに含まれる複数の商品を自動的に判別し、商品ごとに個別のファイルを生成します。
- **Streamlit UI**: ブラウザからCSVファイルをアップロードするだけで、変換されたファイルをZIP形式で一括ダウンロードできます。

## 必要なファイル
- `app.py`: Streamlit Webアプリケーション本体
- `G2828-02　きなこがでろーんTシャツ.xlsx`: 発注書の雛形（テンプレート）
- `requirements.txt`: 実行に必要なライブラリのリスト

## 使い方

### ローカルで実行する場合
1.  Pythonがインストールされていることを確認します。
2.  必要なライブラリをインストールします：
    ```bash
    pip install -r requirements.txt
    ```
3.  Streamlitアプリを起動します：
    ```bash
    streamlit run app.py
    ```
4.  ブラウザで表示された画面から、Shopifyの注文データ（xlsx）をアップロードして「発注書を作成する」をクリックします。

### デプロイ（Streamlit Cloud）
GitHubに本リポジトリをアップロードし、Streamlit Cloudと連携させることで、ブラウザからどこでも利用可能なWebアプリとして公開できます。

## 注意事項
- テンプレートファイル（`G2828-02　きなこがでろーんTシャツ.xlsx`）がプログラムと同じディレクトリに存在する必要があります。
- テンプレートのシート構成やセルの位置（12行目〜18行目など）を変更する場合は、`app.py` 内のマッピング処理を調整してください。
- ファイル名は、SKUと商品名を組み合わせた形式（例：`G2828-01　きなこがでろーんTシャツ.xlsx`）で自動保存されます。
